# 📘 プロジェクト概要

このプロジェクトは、Google BigQueryを活用して野球の試合データをSQLで集計し、その結果をPythonで分析・可視化する実践的なデータ分析例です。特に、**主成分分析（PCA）**を用いて複数のパフォーマンス指標を要約・次元削減し、チームや選手の特徴を直感的に理解することを目的としています。

# 🗂 使用データセット

使用するデータセットは、bigquery-public-data.baseball.games_wide です。これはGoogle BigQuery上で公開されている野球の試合データセットで、チームごとのスコアや勝敗、ヒット数、エラー数などの情報を含みます。

# 🛠 技術スタック
- BigQuery：SQLを用いたデータ抽出・集計
- Python (pandas, numpy)：データ処理と整形
- scikit-learn：主成分分析（PCA）の実施
- matplotlib / seaborn：可視化

# 💡 分析の流れ
- BigQuery上でSQLを使い、games_wideテーブルから必要な指標を集計
- Pythonでデータを取得し、pandas DataFrameに格納
- 主成分分析（PCA）を適用して多変量データの次元を削減
- 各主成分の寄与率・散布図を可視化し、特徴を解釈

# 📈 改善の方向性
- データ標準化：スケールの異なる指標を正規化して分析精度を向上
- 寄与率の確認：主成分が全体の分散をどの程度説明できているか可視化
- 外れ値除去：極端な試合結果が分析を歪めないよう前処理を追加

# 📊 補足
## 主成分分析（PCA）とは？

主成分分析（PCA）は、多くの特徴量を「少数の代表的な軸（主成分）」にまとめることで、データの傾向をシンプルに可視化するための統計的手法です。たとえば野球データでは、「攻撃力」「守備力」「試合展開傾向」といった隠れた要素を主成分として抽出できます。
